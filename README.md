# 🐍 Python Learning Roadmap for Data Engineers

> A structured, hands-on Python course tailored for experienced Java/Data Engineers transitioning to Python.  
> **Estimated time:** ~80 hours over 13 weeks

## Who is this for?

- Senior software engineers with Java background
- Data engineers familiar with Spark, SQL, cloud platforms (AWS/OCI)
- Engineers who want to leverage Python for modern data & AI workflows

## Course Structure

| Phase | Topic | Duration | Notebooks |
|-------|-------|----------|-----------|
| 1 | [Python Core](01-python-core/) | Week 1–2 (~10 hrs) | 4 notebooks |
| 2 | [Data Engineering Essentials](02-data-engineering/) | Week 3–5 (~20 hrs) | 5 notebooks |
| 3 | [Cloud & Infrastructure](03-cloud-infra/) | Week 6–7 (~12 hrs) | 3 notebooks |
| 4 | [Analytics & ML Foundations](04-analytics-ml/) | Week 8–10 (~20 hrs) | 4 notebooks |
| 5 | [Modern Data Stack & AI](05-modern-data-stack/) | Week 11–13 (~18 hrs) | 5 notebooks |

## Getting Started

```bash
# Clone the repo
git clone https://github.com/<your-username>/python-learning-roadmap.git
cd python-learning-roadmap

# Create virtual environment
python -m venv .venv
source .venv/bin/activate  # Linux/Mac
# .venv\Scripts\activate   # Windows

# Install dependencies
pip install -r requirements.txt

# Launch Jupyter
jupyter lab
```

## Legend

Throughout the notebooks you'll see:

- ☕ **Java parallel** — explains the concept by comparing to Java
- 🏋️ **Exercise** — hands-on coding task
- 💡 **Tip** — pythonic best practice
- ⚡ **Skip if you know** — sections you can skim with your background
